Introduction to Python

Learn one the most popular programming languages

Our Introduction to Python course is designed to help teach students who are new to programming with one of the most popular languages. Python can be used for virtually anything and is well supported. Our course guides students through understanding how to program without any of the hassle that comes with figuring it out on our own.

The class will meet once weekly on Wednesdays. See each lesson below for information on particular dates. These sessions will include information, discussion, and time for students to create their own programs based on what they want.

Prerequisites

Students will need a discord account as we will be communicating most of our important information through the platform. Our lessons will take place on discord.

Students will need internet access during our course because our lessons will be live. If they wish to verbally ask questions, they will need a working mic, though it is not a requirement since they can still type their questions.

Rather than download software, students will use repl.it, a free online browser IDE. We will show students how to set it up in the course, though directions are available here.

Introduction to programming and Python

Lesson 2: Variables

Learn how to use variables in Python

Learn variables, a useful and important part of programming. You will learn how to define, manipulate, and use your variables.

Lesson 3: Console I/O and String Manipulation

Learn how to use the console in Python

Learn how to use the console, a way to input and output information with Python. You learn how to manipulate strings, one of the most important data types in Python.

Lesson 4: Basic Operations

Learn how to use basic operations in Python

In this lesson, you will learn arithmetic operations. Arithmetic operations are used to do mathematical calculations and are used in almost every program. These can range from simple addition and subtraction to exponential calculations. We will look at how to perform such operations in Python, using functions and operators as needed.

Next session: 4/06/22

Lesson 5: Conditionals (If/Else)

Learn about programming with branches

In this lesson, we will learn how to implement conditionals in Python. Conditionals are one of the most important control structures you will learn, and will help make your programs more interactive. You will learn how to write if-statements, boolean expressions, and logic operators.

Next session: 4/13/22

6

Lesson 6: Control Structures (Loops)

Learn about looping in Python

In this lesson, we will look at a second type of control structure: the loop. As you become more experienced in coding, you will see that most of your programs will include loops in some form. These can range from simple loops used for iterating through a range of numbers to complex nested loops used to work with complicated data. Throughout this lesson, you will learn about the numerous loop types that Python supports.

Next session: 4/20/22

Lesson 7: Lists

Learn about lists in Python

In this lesson, you will learn one of the most powerful and commonly used data types in Python, the list. Lists are used to store collections of data. You will learn how to define and manipulate lists.

Next session: 4/27/22

8

Lesson 8: Functions

Learn how to write your own functions in Python

You have learned many functions in past lessons. Now, you will learn how to write your own functions. In programming, functions are blocks of reusable code that are used to perform a certain as coded by a user.

Next session: 5/18/22

Lesson 9: Debugging

Learn how to debug the code you write

Now that you've learned the basics of Python, you will learn how to debug code and solve problems in your program. You will learn how to use resources on the internet and how to understand error messages.

Next session: 5/25/22

Lesson 10: Problem Solving
