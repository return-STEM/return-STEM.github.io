Applications of Python

Learn how to create data structures and algorithms in Python with ease

After learning the basics of Python, students can take this intermediate level course to enrich and develop their programming skills. This course traverses a wide range of topics, focusing particularly on creating data structures and algorithms with the object oriented programming paradigm. Students will also learn key libraries and modules in python to create applications of increasing complexity.

The class will meet once weekly on Thursdays. See each lesson below for information on particular dates. These sessions will include time for students to solve programming problems and learn essential programming skills.

Prerequisites

Students who take this course should have taken an Introductory course in Python or similar. It is also recommended that they have taken at least an algebra level math course.

Students will need internet access during our course because our lessons will be live. If they wish to verbally ask questions, they will need a working mic, though it is not a requirement since they can still type their questions.

Rather than download software, students will use repl.it, a free online browser IDE. We will show students how to set it up in the course, though directions are available here.

Lesson 1: Review from Intro to Python

Review of introductory Python concepts

You will review basic concepts that are covered in an introductory course, to refresh your memory or to fill in gaps in your knowledge. You will use the concepts covered in this review in this course.

Next session: 3/17/22

2

Lesson 2: Object-oriented Programming

Introduction to object-oriented programming

This lesson covers the basic idea and principles of object-oriented programming. You will learn concepts related to object-oriented programming, such as encapsulation, inheritance, abstraction, and polymorphism.

Next session: 3/24/22

3

Lesson 3: Defining a Class

Learn how to write your own classes

In this lesson, you will learn how to implement and write classes. You will be able to represent objects in the real world with classes and implement them with attributes and methods.

Next session: 3/31/22

4

Lesson 4: Inheritance and Polymorphism

Learn how to implement inheritance and polymorphism

In this lesson, you will learn inheritance and polymorphism, and how they are implemented in the classes you will write. You learn how to use them to improve the code that you write.

Next session: 4/07/22

5

Lesson 5: Algorithmic Thinking

Learn the process of algorithmic thinking and improve your problem solving skills

In this lesson, you will learn the implementation of common algorithms such as binary search. You will learn the process of developing an algorithm and Big O notation.

Next session: 4/14/22

6

Lesson 6: Problem Day

Practice the skills you have learned

Students will use the knowledge they have learned to solve short problems with mentor guidance, to clear up confusions and strengthen up their knowledge.

Next session: 4/21/22

7

Lesson 7: Linked Lists and other data structures

Learn how to use different data structures and the benefits of using them

Students learn how to create a node class and learn how to create linked lists from the node list, like headers and append functions. Students also learn other important data structures and how to use them.

Next session: 4/28/22

8

Lesson 8: Function-function interactions

Advance your knowledge of functions in python and how to expand their uses

Students will learn how functions can be used as parameters and decorators. They will learn how variable defined names can represent functions.

Next session: 5/19/22
