Introduction to C++

Learn one of the world's most powerful programming languages

Begin your journey into the world of C++, a popular and powerful languages used for various applications including game development, virtual reality, and robotics. C++ allows students to learn and understand the intricacies of the computer while still controlling it all from a high level environment. The course explores everything needed for students to build their own console applications and more.

The class will meet once weekly on Mondays. See each lesson below for information on particular dates. These sessions will give students the instruction and time to create C++ programs.

Prerequisites

Students will need a discord account as we will be communicating most of our important information through the platform. Our lessons will take place on discord.

Students will need internet access during our course because our lessons will be live. If they wish to verbally ask questions, they will need a working mic, though it is not a requirement since they can still type their questions.

Rather than download software, students will use repl.it, a free online browser IDE. We will show students how to set it up in the course, though directions are available here.

1

Lesson 1: Welcome to C++

Introduction to the C++ language and its semantics

This lesson will consist of an introduction to the C++ language and explanations of how code is run and what happens within the computer. By the end, you will understand basic structure/syntax of C++ and you should be able to write a simple program.

Next session: 3/14/22

2

Lesson 2: Variables and Data Types

Explore how C++ stores information

In this lesson, we will look at variables, a useful and integral part of the of programs. We will learn how to define and use our own variables, and how we can manipulate the values stored within them. We will also learn how C++ handles variables of different data types.

Next session: 3/21/22

3

Lesson 3: Console I/O

Understand the C++ console input and output

Sometimes, we may want to input information from the console a certain way or need a way to output information in a certain format. This lesson will go over all the intricacies of taking input and output from the console using C++.

Next session: 3/28/22

4

Lesson 4: Arithmetic Operations

Learn how to use arithmetic operations in C++

In this lesson, we will look at arithmetic operations. Arithmetic operations are used to do mathematical calculations and are used in almost every program. These can range from simple addition and subtraction to exponential calculations. We will look at how to perform such operations in C++, using functions and operators as needed.

Next session: 4/04/22

5

Lesson 5: Conditionals

Incorporate decision making to programs

In this lesson, we will learn how to implement conditionals in C++. Conditionals are one of the most important control structures you will learn, and will help make your programs more interactive. You will learn how to write if-statements, switch statements, boolean expressions, and logical operators.

Next session: 4/11/22

6

Lesson 6: Looping

Learn to write less code and do more with loops

In this lesson, we will look at a second type of control structure: the loop. As you become more experienced in coding, you will see that most of your programs will include repeated code. Loops allow for a way to reuse the same block of code in different ways. We will also learn how you can nest loops to create more complex programs. Throughout this lesson, you will learn about the numerous loop types that C++ supports.

Next session: 4/18/22

7

Lesson 7: Arrays

Arrays let you store more data in one variable

In this lesson, you will learn how to use arrays. You learn how to define and manipulate them, along with related libraries. Arrays are used to represent a collection of the same data type that belong together. By the end of the lesson, you will understand the uses and limitations of arrays in C++

Next session: 4/25/22

8

Lesson 8: Functions

Organize and reuse code with functions

You have learned many functions in past lessons. Now, you will learn how to write your own functions. In programming, functions are blocks of reusable code that are used to perform a certain as coded by a user.

Next session: 5/16/22

9

Lesson 9: C++ Standard Library

Explore new and powerful features of the C++ language

The C++ language comes with many powerful tools for writing programs. In this lesson we will go over some commonly used libraries in the C++ standard and explain how to read and understand documentation of them. These include <string>, <vector>, <algorithm>, and more.

Next session: 5/23/22

10

Lesson 10: Debugging

Find and squash the bugs in your program

After writing many programs, you are bound to find some problems that you just don't know how to fix. This lesson goes over some of the techniques to find the bugs in your program and how to use the C++ debugger. It also includes a brief introduction to exceptions in C++.

Next session: 5/30/22

11

Lesson 11: Problem Solving

Prepare to jump off on your own C++ adventure

This lesson goes over how to think about solving programming challenges and translating what you want to do into code. The final lesson of our Introduction to C++ course will make sure you have all the tools needed for you to start projects of your own.

Next session: 6/06/22
