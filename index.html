Introduction to Python

  Beginner

10 Weeks

Introduction to Python

  • Our Introduction to Python course is designed to teach middle school and high school students how to create useful applications as they see fit, even without prior programming knowledge.
  • Students will meet with mentors once weekly for a one hour session over ten weeks.
  • Students can choose different timings as fits their schedule.
  • These sessions will include information, discussion, and time for students to create their own programs based on what they want.

Course Layout Information

  • Lessons 1-3 go over how programming works, variables types, and console IO/ manipulation.
  • Lessons 4-6 go over mathematical operations in programming, conditional logic, and control structures.
  • Lessons 7-9 go over data structures, functions, and methods to debug python code.
  • Lesson 10 goes over problem solving with programming
  • After the course is a 2 day mini-hackathon with workshops and prizes.

Course Prerequisites

  • Students will need a discord account as we will be communicating most of our important information through the platform. Our lessons will take place on discord.

Introduction to C++

  • Learn the basics of programming with one of the most powerful programming languages. C++ is used in a wide variety of applications, such as game and software development, VR, and robotics.
  • Over the course of 11 weeks, students will meet with mentors once per week for one hour to receive remote instruction.
  • By the completion of this course, students will be able to make their own console applications.

Applications of Python

  Moderate

8 Weeks

Applications to Python

  • After learning the basics of python, students can take this intermediate course to enrich and develop their skills.
  • Over the course of 8 weeks, students will learn the fundamentals of object oriented programming and how to design data structures and algorithms.
  • Students will learn various useful libraries and modules, and be able to build applications with increasing complexity.
  • Students will be able to solve algorithm and data structure problems with mentor guidance.

Course Layout Information

  • Lesson 1 reviews fundamentals from Introduction to Python necessary to learn this course.
  • Lessons 2-4 cover important concepts of Object Oriented Programming and the class semantics of Python.
  • Lessons 5 and 6 will cover important algorithms to help students improve their problem solving skills.
  • Lesson 7 teaches students the creation and application of linked lists, as well as how to integrate them into classes.
  • Lesson 8 covers how functions can be used as parameter and decorator functions, and how variable names can represent functions.
